Действия с командами:

Создавать команду

У владельца:
Редактировать команду
Добавлять, удалять мемберов
Изменять у мемберов статус
Передавать роль владельца

У модераторов:
Добавлять мемберов

У всех:
Просматривать список участников
Выйти из команды (кроме владельца)


Создание команды
POST /u/teams/



Просмотреть информацию по команде (у всех)
GET /u/teams/{name}/

Передать роль владельца (у овнера)
POST /u/teams/{name}/transfer_ownership/

Редактировать команду (только овнер)
PUT PATCH /u/teams/{name}/

Посмотреть участников команды (у всех)
GET /u/teams/{name}/m/

Добавить участника (только у модератора и овнера)
POST /u/teams/{name}/m/

Редактировать и удалиь мембера (только овнер)
POST PUT PATCH DELETE /u/teams/{name}/m/{name}
