Действия с командами:

Создавать команду

У владельца:
+ Редактировать команду
+ Добавлять, удалять мемберов
+ Изменять у мемберов статус
+ Передавать роль владельца

У модераторов:
+ Добавлять мемберов

У всех:
+ Просматривать список участников
+ Выйти из команды (кроме владельца) через delete
